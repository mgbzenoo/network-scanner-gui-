# Network Scanner GUI

A beginner-friendly Python GUI for running Nmap scans.

## Features
- Input target IP or domain
- Run basic Nmap scan (`-Pn`)
- Display results in a scrollable text box

## Requirements
- Python 3.x
- Nmap installed on your system
- Tkinter (included with most Python installs)

## How to Run
```bash
pip install -r requirements.txt
python gui.py
